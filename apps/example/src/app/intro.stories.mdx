# Metad Formly Components

Let's define a story for our `Formly` components

## Material Components

```javascript
{
  key: 'value',
  type: 'json',
  templateOptions: {
    label: 'JSON'
  }
}
```

<Story name="1. Textarea for JSON Object">
  {{
    template:`<metad-formly [schema]="[{
      key: 'value',
      type: 'json',
      templateOptions: {
        label: 'JSON'
      }
    }]"
    [model]="{
      value: {
        key: 'value',
        type: 'json',
        templateOptions: {
          label: 'JSON'
        }
      }
    }"></metad-formly>`,
  }}
</Story>

You can set attributes in templateOptions to change material component appearance:

```javascript
[
  {
    key: "value",
    type: "json",
    templateOptions: {
      label: "JSON Outline",
      attributes: {
        appearance: "outline",
      },
    },
  },
  {
    key: "value2",
    type: "json",
    templateOptions: {
      label: "JSON Fill",
      attributes: {
        appearance: "fill",
      },
    },
  },
]
```

Result:
